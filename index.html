custom google maps

get started

For have a Custom Google Maps takes only these four steps.
  • Include script google API https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY
  • Include plugin Custom Google Maps map.js
  • Include a basic markup HTML map: <div id="wrapperIp" data-pin-lat="-23.562353" data-pin-lng="-46.503113" data-map-lat="-23.562353" data-map-lng="-46.503113"></div>
  • Initialize the map with Contructors like its down.

Examples

Exemplo 1

var exampleOne = new Map({ 'name': 'exampleOne' ,'wrapperId':'exampleOne' ,'zoom': 12 ,'mobile': false });
